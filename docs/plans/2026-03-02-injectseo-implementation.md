# InjectSEO Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build the InjectSEO med spa SEO agency platform — a Next.js app that serves as both the public agency website and an internal AI-powered sales toolkit.

**Architecture:** Next.js 15 App Router with two route groups: `(public)` for the agency website and `(dashboard)` for internal AI tools behind auth. Public pages are static/SSG for speed. Dashboard uses API routes that call Claude API for AI features and Supabase for lead persistence. PDF generation via `@react-pdf/renderer`. Site analysis via Google PageSpeed Insights API + HTML parsing.

**Tech Stack:** Next.js 15, React 19, Tailwind CSS 4, shadcn/ui, Prisma 6 + Supabase (free tier), Claude API (`@anthropic-ai/sdk`), `@react-pdf/renderer`, Google PageSpeed Insights API, Google Places API, Calendly embed.

**Design Reference:** See `docs/plans/2026-03-02-injectseo-agency-design.md` for full brand spec.

**Priority Order:** Website first (so prospects can check you out), then DM Generator + SEO Audit (critical for outreach), then remaining tools.

## Phase 4: Authentication

### Task 17: Simple Dashboard Auth

**Files:**
- Create: `src/app/(dashboard)/login/page.tsx`
- Create: `src/app/api/auth/route.ts`
- Create: `src/lib/auth.ts`
- Create: `src/middleware.ts`

**Step 1: Create auth API route**

`src/app/api/auth/route.ts` — POST endpoint. Accepts `{ password }`. Compares against `process.env.DASHBOARD_PASSWORD`. If match, sets an httpOnly cookie `injectseo-auth` with a signed JWT (use `jose` library — `pnpm add jose`). Returns 200 or 401.

**Step 2: Create auth utility**

`src/lib/auth.ts` — `verifyAuth(request)` function that reads the cookie, verifies the JWT, returns boolean.

**Step 3: Create middleware**

`src/middleware.ts` — protect all `/dashboard/*` routes. Check for valid auth cookie. If missing/invalid, redirect to `/dashboard/login`.

```typescript
import { NextRequest, NextResponse } from "next/server";
import { jwtVerify } from "jose";

export async function middleware(request: NextRequest) {
  if (request.nextUrl.pathname.startsWith("/dashboard")) {
    if (request.nextUrl.pathname === "/dashboard/login") return NextResponse.next();

    const token = request.cookies.get("injectseo-auth")?.value;
    if (!token) return NextResponse.redirect(new URL("/dashboard/login", request.url));

    try {
      const secret = new TextEncoder().encode(process.env.DASHBOARD_PASSWORD);
      await jwtVerify(token, secret);
      return NextResponse.next();
    } catch {
      return NextResponse.redirect(new URL("/dashboard/login", request.url));
    }
  }
  return NextResponse.next();
}

export const config = { matcher: ["/dashboard/:path*"] };
```
